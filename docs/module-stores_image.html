(static) uploadImage() → {Promise.<void>}

Description:
  • Uploads an image to Firebase Storage and saves it under a batch in Firestore. All images uploaded at the same time share a batchId, so they're grouped together. We use setDoc with merge:true on the batch so we don't overwrite it when uploading multiple files.
