useSortAndFilter(items, searchQuery, sortOrder) → {Object}

Takes a list of items and returns it sorted depending on what has been chosen. Filtering checks if the item's title includes the search query (case-insensitive). Sorting can be: 'newest', 'oldest', 'alphabetical', or 'alphabetical-reverse'. Note: newest/oldest sorting uses uploadDate.seconds because Firestore timestamps aren't plain JS.
Parameters:
Name Type Description
items Array.<Object> The list to work with.
searchQuery string What the user typed in the search bar.
sortOrder string | null The chosen sort option, or null for no sorting.
